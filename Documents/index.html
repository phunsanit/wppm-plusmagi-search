Why PlusMagi Search?

🔒

Strict Access Control

Public users see published content. Editors see everything. Smart filtering based on user roles and capabilities.

⚡

Deep Search

Search beyond titles. Include custom fields (meta), categories, and tags in your search results automatically.

🏷️

Smart Prefixes

Power users can use prefixes like post:, tag:, or category: to filter results instantly.

🎨

Rich UI & Tabs

Results are organized into tabbed categories (Posts, Tags, Categories) with thumbnail support.

🚀

High Performance

AJAX-powered and lightweight. Designed to be fast with no bloat or external dependencies.

🧩

Easy Integration

Drop the [plusmagi_search] shortcode anywhere or use the built-in Gutenberg Block.

Get Started in Seconds

Installing PlusMagi Search is as simple as any other WordPress plugin.

1 Upload & Activate

Upload the plugin folder to your /wp-content/plugins/ directory and activate it.

2 Add Shortcode

Place the shortcode [plusmagi_search] in any page or post.

3 Enjoy!

Your secure, advanced search is now live.
